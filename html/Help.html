Configuration

  • Open a landscape: Select File>Open Landscape ... and browse to a landscape. Select the landscape and then New Configuration.
  • Open an existing configuration: Select File>Open Landscape ... and browse to a landscape. Select a landscape, the configuration from the list at the bottom of the dialog, and then Open.
  • Remove a configuration: Select File>Delete Configuration.
  • Save the current configuration: Select File>Save Configuration or File>Save Configuration As.

Viewpoint
  • Translate the viewpoint along a plane parallel with the terrain: Place the cursor over the terrain, press the left mouse button, and drag. The terrain movement will die off gradually after the mouse release. Press any mouse button to stop the movement immediately.
  • Translate the viewpoint along the plane of the screen: Place the cursor over the terrain, press the middle mouse button, and drag.
  • Rotate the viewpoint about the center of the view: Place the cursor over the terrain, press the right mouse button, and drag.
  • Dolly the viewpoint in and out from the center of the view: Move the mouse scroll wheel.
  • Set the center of rotation: Translate the viewpoint until the CoR cross-hair is in the desired location. Alternatively, single-click with the right mouse button at the desired location and select Set Center of Rotation from the menu.
  • Change magnification (zoom) of the view: Press the zoom mode button. Use the mouse scroll wheel to change the magnification level. Press the zoom button again to go back to dolly mode.
  • Set the viewpoint to the home position: Press the home button (overhead view, center of rotation at landscape center, magnification = 1).
  • Open the viewpoint list: Press the viewpoint button and then select Open Viewpoint List.
  • Add the current viewpoint to the list of saved viewpoints: Open the viewpoint list and press "+".
  • Remove a viewpoint from the list of saved viewpoints: Open the viewpoint list and select the viewpoint from the list. Press "-".
  • Move to the previous viewpoint in the saved list: Open the viewpoint list  and press the left arrow.
  • Move to the next viewpoint in the saved list: Open the viewpoint list and press the right arrow.
  • Edit a saved viewpoint: Select the viewpoint from the list and change the fields (retrieve the current viewpoint with the Current button). Press Save.

Marble
  • Place the green marble somewhere in the scene: Single-click with the left mouse button at the desired location. Alternatively, type the coordinates in the marble location field at the top right of the worldview and press return.
  • Open the marble info view: Press the marble button in the toolbar.
  • Find the coordinates at a given point: Single-click at the desired location.  See the coordinates in the marble location field. Additional information can be found in the marble view.

Map Elements
  • Add a map element (Placemark, Figure, Billboard, Path, Plane, Grid, Camera, Profile, or Scale) to the scene: Right-click and select Add>map element type from the context menu. You may also open the map elements view and press the appropriate Add button at the bottom.  Hover the cursor over a button to identify it.
  • Delete a map element: Right-click on the element and select Delete from the context menu. The element may also be deleted from the map elements view.
  • Hide a map element: Right-click on the element and select Hide from the context menu. The element may also be hidden from the map elements view.
  • Edit a map element: Right-click on the element and select Edit from the context menu. The element may also be edited from the map elements view.
  • Rename a map element: Right-click on the element and select Rename from the context menu. The element may also be renamed from the map elements view.
  • Move a map element: Place the cursor over the map element, press the left mouse button down and drag.

Profile
  • Adjust the length of the profile line: Place the cursor over one of the profile paddles, press the left mouse button down and drag.
  • Find the value of a point on the profile graph: Single-click with the left mouse button at the desired location in the graph view.
  • Open a closed profile graph: Open the Map Elements view and select the profile from the list. Press Open.

Tape Measure
  • Measure the distance, slope, or height between two points: Place DERT in measuring mode by selecting the tape measure button. Single-click at the location of the first point with the left mouse button, then single-click at the location of the second point.  Read the values from the tape measure window.

Scene Attributes
  • Change the color of the sky: Select Edit>Change Background Color to display the background dialog.  Select a predefined color, or use the color chooser.  Press OK.
  • Change to stereo viewing: Select Edit>Stereo. Select the Enable Stereo check box. Press OK.
  • Show/Hide the center of rotation cross hair: Select Edit>Landscape>Show Crosshair at Center of Rotation.
  • Show/Hide the center scale: Select Edit>Show Center Scale Overlay.
  • Show/Hide the marble: Select Edit>Show Marble.
  • Show/Hide the text overlay: Select Edit>Show Text Overlay.
  • Bring all Map Elements to front: Select Edit>Map Elements On Top.
  • Show hidden lines as dashes: Select Edit>Show Tool Hidden Lines As Dashes.
  • Change WorldView canvas dimensions: Select Edit>Set WorldView Canvas Dimensions.
  • Change Map Element preferences: Select Edit>Edit Map Element Preferences.

Lighting and Shadows
  • Switch between main light modes: Open the lighting and shadows view. Press Solar or Artificial.
  • Enable/disable the headlight: Open the lighting and shadows view. Press Enable in the headlight section.
  • Enable/disable shadows: Open the lighting and shadows view. Press Enable in the Shadows section. Use the light positioning view to change shadows.
  • Set the center and radius of the shadow sphere (area covered by shadows): Open the lighting and shadows view and enter the values in the Shadows section (Note: You can copy and paste from the marble location field or other window).
  • Change the position of the light: Open the light positioning view. Adjust the position via the calendar or the az/el controls.
  • Change light properties: Open the lighting and shadows view and enter the values in the main light or headlight sections.

Surface
  • Change the elevation vertical exaggeration: Open the surface and layers view. Change the Vert Exaggeration field.
  • Display the terrain as wireframe: Open the surface and layers view. Select Wireframe.
  • Show/Hide the terrain surface normals: Open the surface and layers view. Select Surface Normals.
  • Change the surface color of the terrain: Open the surface and layers view. Press the Surface Color button.
  • Display a grid draped on the terrain surface: Open the surface and layers view. Select Surface Grid. Set Cell Size and Grid Color.
  • Display surface shading: Open the surface and layers view. Select Surface Shading.

Layers
  • Show/Hide the layers: Open the surface and view. Deselect Show Layers in the Layers section.
  • Display an additional layer: Open the surface and layers view. Press Configure. Select the destination layer in the left list. Select the layer from the right list and move it to the left list using the arrows. Press OK.
  • Display an elevation, slope, or aspect map: Open the surface and layers view and add the desired layer. Press OK.
  • Set a color map for an elevation, slope, or aspect map: Open the color bars view and press Color Map for the layer. Set the color map and parameters in the dialog.
  • Change the opacity of a layer: Open the surface and layers view.  Change the opacity spinner next to a layer to set the percentage.  Lock a layer (press lock icon) to stop its auto-adjustment.
  • Distribute the opacity of each layer equally: Open the surface and layers view and press Distribute.

Camera
  • Point the camera: Click at the desired location in the landscape. Copy the coordinates from the marble location field and paste them in the point at field in the camera view. Press Point At.
  • Open a closed camera view: Open the map elements view and select the camera from the list. Press Open.
